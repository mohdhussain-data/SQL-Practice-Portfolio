# SQL Data Analysis Portfolio

This repository contains structured SQL analysis built using the **Parch & Posey sales dataset**, a relational database representing a B2B paper company’s sales operations.

The project demonstrates practical SQL techniques used in real-world data analysis including data exploration, business metric calculation, relational analysis, data cleaning, and advanced analytical queries.

The repository is organized as a progressive SQL workflow that moves from foundational querying to advanced analytical techniques and business-focused analysis.

---

# Dataset

The analysis is based on the **Parch & Posey dataset**, which simulates the operations of a B2B company selling paper products to corporate clients.

The dataset includes the following relational tables:

- accounts – customer companies  
- orders – purchase transactions  
- sales_reps – sales representatives responsible for accounts  
- region – regional sales structure  
- web_events – marketing and website interaction activity  

A small mock table **sf_crime_data** was also created to demonstrate datetime parsing and data cleaning techniques.

# SQL Skills Demonstrated

The queries in this repository demonstrate analytical SQL techniques commonly used in data analyst and reporting roles.

Core capabilities include:

- Data retrieval and filtering  
- Aggregations and metric generation  
- Multi-table relational analysis using joins  
- Subqueries and nested analytical logic  
- Common Table Expressions (CTEs)  
- Data cleaning and transformation  
- Window functions for advanced analytics  
- Ranking, running totals, and percentiles  
- Combining datasets using UNION  
- Self joins for event sequence analysis  
- Query structure optimization and performance considerations

# Repository Structure

The project is organized into progressive SQL modules.
01_Basics.sql
Core query structure including SELECT, WHERE, ORDER BY, and LIMIT.
02_Aggregations.sql
Metric generation using aggregate functions and grouped analysis.
03_Joins.sql
Relational analysis across multiple tables.
04_Subqueries.sql
Multi-step analysis using nested queries.
05_Mixed_Problems.sql
Business-focused SQL problem solving combining multiple techniques.
06_CTEs.sql
Structured multi-step queries using Common Table Expressions.
07_Data_Cleaning.sql
Transforming raw text and datetime fields into structured analytical data.
08_Window_Functions.sql
Advanced analytical techniques including running totals, ranking, lag analysis, and percentiles.
09_Advanced_SQL.sql
Advanced SQL patterns including UNION, SELF JOIN, FULL OUTER JOIN, and query optimization techniques.
10_Sales_Analysis_Project.sql
Business analysis of customer activity, sales performance, and revenue trends.

Each SQL file documents the analytical process including:

- business question  
- query planning and logic  
- final SQL implementation
